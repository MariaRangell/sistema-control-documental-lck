## 🎯 Descripción

Sistema desarrollado para **LCK Consultores** que permite cargar, organizar y administrar documentación interna de la empresa. Cuenta con autenticación por login y acceso segmentado a cinco áreas departamentales.

**Áreas del sistema:**
- 🏢 Empresa
- 👥 Recursos Humanos
- 🔍 Auditoría
- 🤝 Clientes
- 📦 Proveedores

---

## ✅ Funcionalidades

- Autenticación con login por usuario y contraseña
- Carga y administración de documentos por área
- Acceso segmentado según el área del usuario
- Gestión de información interna por departamento
- Recuperación de contraseña con notificación al administrador

## ⚙️ Instalación local

```bash
git clone https://github.com/MariaRangell/sistema-control-documental-LCK-CONSULTORES.git
cd sistema-control-documental-LCK-CONSULTORES

# Backend
cd backend
npm install
npm run dev

# Frontend
# Abre index.html en tu navegador o usa Live Server
```
